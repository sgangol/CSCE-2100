Welcome to the Double-Linked-List Program

Compile: Type "make" in the terimnal to compile

Run: Type "make run" in the terminal after compile

Menu: 

1. Append
2. InsertAt
3. Remove
4. RemoveAt
5. Search
6. Print
7. Quit

INSTRUCTION: To use this program, Choose the input of the <function> 
from table above hit <space> and type in the <character> or/and <input> 
seperated by space when applicable

Example:
APPEND
CSCE2100> 1 A  		<- ADDS A TO THE LINKED LIST

INSERTAT
CSCE2100> 2 A 2		<- ADDS A TO THE SECOND POSITION*

REMOVE
CSCE2100> 3 A 		<- REMOVES A* 

REMOVEAT
CSCE2100> 4 A 2		<- REMOVES A FROM 2 POSITION*

SEARCH
CSCE2100> 5 A		<- SEARCHES FOR A 

PRINT
CSCE2100> 6		<- PRINTS THE LIST BOTH WAYS

QUIT
CSCE2100> 7		<- QUITS THE PROGRAM

* if applicable
